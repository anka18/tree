### 1. Управление проектом

#### 1.1 Сборка проекта

```
docker-compose build
```

#### 1.2 Запуск проекта
В интереактивном режиме
```
docker-compose up
```
В фоновом режиме
```
docker-compose up -d
```

Проект будет доступен по адресу http://127.0.0.1:9090

#### 1.3 Остановка проекта
```
docker-compose down
```
